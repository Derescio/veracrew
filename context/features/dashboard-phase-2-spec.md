# Dashboard UI Phase 2 Spec

## Overview

This is phase 2 of 3 for the dashboard UI layout. Use the screenshot referenced below, only make improvements to the UI if necessary. Use the data from the mock data file referenced below, if that has already been done then move on. Add dome colors to the icons and refrence those colours on the dashboard by using a side border with the same color.

## Requirements for phase 2

- Collapsible sidebar(on large screens, confirm mobile is ok) 
- Drawer icon to open/close sidebar (maybe left and right chevron)
- Items/types with links to /items/TYPE (eg.items/snippets)
- Favorite collections
- Most recent collections
- User avatar area at the bottom as well as settings icon
- Always a drawer on mobile view
